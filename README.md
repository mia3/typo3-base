# TYPO3 CMS Base Distribution

Get going quickly with TYPO3 CMS.

## Prerequisites

* PHP 7.2
* [Composer](https://getcomposer.org/download/)

## Quickstart

* `composer create-project mia3/typo3-base project-name --stability=dev`
* `cd project-name`
* `git init`
* `git add .`
* `git commit -a -m "initial commit"`
* `make setup`
